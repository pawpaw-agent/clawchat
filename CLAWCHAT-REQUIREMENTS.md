# ClawChat 项目需求文档

> OpenClaw 移动端控制台 — 在安卓手机上使用和管理 OpenClaw

---

## 项目概述

**ClawChat** 是一个移动端 Web 应用，让用户可以在本地网络中通过安卓手机访问和管理 OpenClaw 网关。

### 核心价值
- 📱 **移动优先**: 专为安卓手机优化的界面
- 🏠 **本地网络**: 无需公网，保护隐私
- 🔌 **双模式集成**: WebChat 聊天 + Control UI 管理
- 🤖 **AI 切换**: 快速切换不同 AI 智能体

---

## 目标用户

- OpenClaw 个人用户
- 需要在手机上管理 Agent 的开发者
- 本地网络环境下的移动办公场景

---

## MVP 功能列表

### P0 - 核心功能 (必须有)

| ID | 功能 | 描述 | 验收标准 |
|----|------|------|----------|
| F1 | 网关连接 | 连接本地 OpenClaw 网关 | 可配置 IP:端口，显示连接状态 |
| F2 | WebChat 聊天 | 与 AI 智能体对话 | 发送/接收消息，支持 Markdown |
| F3 | AI 切换 | 切换不同智能体 | 下拉选择，切换后对话上下文独立 |
| F4 | 会话列表 | 查看历史对话 | 按时间排序，可删除会话 |
| F5 | Control UI 概览 | 查看网关状态 | 显示网关状态、Agent 列表、会话数 |

### P1 - 增强功能 (应该有)

| ID | 功能 | 描述 |
|----|------|------|
| F6 | 快捷命令 | 常用 OpenClaw 命令快捷按钮 |
| F7 | 消息搜索 | 在历史消息中搜索 |
| F8 | 深色模式 | 自动/手动深色主题 |
| F9 | 离线缓存 | 缓存最近消息 |

### P2 - 未来功能 (可以有)

| ID | 功能 | 描述 |
|----|------|------|
| F10 | 推送通知 | 新消息通知 |
| F11 | 语音输入 | 语音转文字 |
| F12 | 多设备同步 | 跨设备会话同步 |

## 技术需求

### 前端
- **框架**: React + Vite (轻量、快速)
- **UI**: TailwindCSS + HeadlessUI (移动端友好)
- **状态管理**: Zustand (简单轻量)
- **PWA**: 支持安装到主屏幕

### 后端
- **模式**: 纯前端，直接调用 OpenClaw Gateway API
- **协议**: WebSocket (聊天) + REST (管理)
- **认证**: Token 认证 (与 OpenClaw 一致)

### 部署
- **构建**: Vite build → 静态文件
- **托管**: 可部署到任意静态服务器
- **访问**: 本地网络 IP 访问

## 非功能需求

| 类别 | 要求 |
|------|------|
| **性能** | 首屏加载 < 2s，消息延迟 < 200ms |
| **兼容** | Android Chrome 90+, iOS Safari 15+ |
| **安全** | 仅本地网络，Token 本地存储 |
| **可用性** | 离线可用 (PWA)，弱网优化 |

---

## 项目约束

- ✅ 仅支持本地网络访问
- ✅ 不需要后端服务
- ✅ 与 OpenClaw Gateway 直接通信
- ✅ 支持 OpenClaw 2026.3.x 版本
